# Métodos reutilizáveis

Metodologia **genérica** para dossiês brand safety / vetting — aplica-se a **qualquer cliente**, não só Loterias.

| Método | Arquivo |
|--------|---------|
| Descoberta de perfis (@) | [`discovery/METODO_DESCOBERTA_PERFIS.md`](discovery/METODO_DESCOBERTA_PERFIS.md) |
| Brand safety / OSINT | [`brand-safety/METODO_BRAND_SAFETY.md`](brand-safety/METODO_BRAND_SAFETY.md) |
| Tabela de evidências (exemplo) | [`brand-safety/FONTES_BRAND_SAFETY_LOTERIAS2026.md`](brand-safety/FONTES_BRAND_SAFETY_LOTERIAS2026.md) |

Pesquisa **específica de um lote** fica em `projects/<cliente>/…/research/`.

**Como usar:** leia o método antes da coleta; registre evidências em `research/` do projeto ou em arquivo FONTES dedicado. O HTML entregue ao cliente **não** deve citar estes caminhos.
